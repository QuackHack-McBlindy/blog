Wake Word

This section is about Wake Word detection.

No tflite-runtime

I really wanted to go my own route here.
But since there is no tflite-runtime in the nixpkgs, and I was unable to package it myself,
I had make multiple workarounds and ended up using the wyoming-openwakeword package.
There is nothing wrong with that, it's working well, but the workflow does look a little weird.
I had to create a systemd service that reads the output from the wyoming-openwakeword service.
It's all described pretty well in the link below.
